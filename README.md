# Cs325-Project
**Description**
This project is for credit card fraud detection using three different classification models. For our model training/testing data, we are using Machine Learning Group's Credit Card Fraud Detection dataset, which has 284,807 transaction datapoints. (link here:https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud). Due to confidentiality concerns, data has been preprocessed with PCA in order to maintain user anonimity due to this data being from real European transactions in 2013.

**Installation**
To install the project, run requirements.txt in terminal and download Machine Learning Group's credit card fraud dataset from the link above. Then, run each notebook to get our results, random seed is kept at random_seed=42 so results stay consistent across users.

**Usage**
This project uses sklearn's machine learning library to initialize, validate, and test the models.  The project utilizes jupyter notebooks so results can be seen and run within those.
